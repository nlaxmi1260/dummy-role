Ansible roles with Demo
  1. Ansible Galaxy - Exploring pre-built ansible roles
    1. in ansible-galaxy there are multiple number of ansible roles which are built in
    2. ansible-galaxy is install with ansible
      $ ansible-galaxy -h
      mainly used for collections,role
      $ ansible-galaxy role -h
      list some role commands
  2. Ansible Galaxy - importing and installing roles
    $ ansible-galaxy role -h
  3. How can we use the Docker ansible roles that is there in the ansible-galaxy
    1. login ansible-galaxy with github account
    2. ansible-galaxy we can loged in
    3. In ansible-galaxy copy the command
      $ ansible-galaxy role install bsmeding.docker
      bsmeding.docker role is installed.To check role where it installed
      $ ls ~/.ansible/roles
    4. After installing the role how we can use the role
      we have to create a playbook - userole.yaml
    5. To run the file
      $ ansible-playbook -i inventory.ini userole.yaml
    6. to check docker is installed or not on target mechine
      $ docker
  4. How to publish roles in the ansible-galaxy
    1. To publish roles in ansible-galaxy
      ansible roles should always be put in github
      Should be imported into the ansible-galaxy
    2. if we have created a role then go to the role
      $ cd httpd
      goto the github and create a new repo - dummy repo
      $ git init
      copy the https url of git repo
      $ git remote add origin https://github.com/nlaxmi1260/dummy-role.git
      $ git remote -v
      $ git add .
      $ git commit -m "initial files"
      $ git push origin main
    3. our role will be pushed to github repo ,now i hv to push ansible-galaxy
      $ ansible-galaxy import githubusername nameofTheRepo
      we will get authenication error
      go to ansible-galaxy -> collections -> api token -> load the token ->copy token
      $ ansible-galaxy import githubusername nameofTheRepo --token tokenId
